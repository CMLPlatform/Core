Button information All buttons correspond to the tree that is selected (e.g. region of consumption). In the case that single selection is in effect, the buttons select shown and select below will only select a single node.

Terminology: the folder icon represents a parent node, the file icon represents a child node. It is possible that a parent node contains sub-parents nodes and this is displayed as a folder icon as well.

Select below: Allows for quick selections of specific sub-parents or children nodes. This button selects every sub parent or child node of the highlighted selection. If a node is highlighted it turns blue. If selected it turns green.

Select shown: Allows for quick selections of many nodes at the same time. This button selects all children nodes, sub-parent or parent nodes that are visible/shown. It selects every node that is currently visible with the exception that it deselects any parent or sub parent node that those nodes may have.

Deselect all: Deselects all nodes whether visible or not.

Expand: Expands one level.

Collapse: Collapse one level.

Collapse all: Collapses all possible node selections to the highest level.

Visualize current & start session:Visualizes current selections and resets the filter selections while starting a session (empty selection canvas, but any selections at that point will be remembered)

Visualize in session:Remembers previous selections in filter

Visualize & reset session:Resets previous selections in filter
