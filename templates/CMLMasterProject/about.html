What is CORE?

The CML platform was developed for scientific communication purposes and the development of several web-based ICT tools (apps). It is developed in such a way that these tools are modular while serving their own content. This helps in software development, as the platform provides simple features to integrate complete web-apps. Apps can be stored under their own domain and cloud, once an app is "production ready". CORE is developed with the Django web framework.
In relation to scientific communication purposes we showcase project developments and an app called "MicroVis" which are sets of interactive visualisations in order to communicate scientific results.
