Introduction Educational Materials

The aim of CircuMat project is to support the introduction of Circular Economy concepts to practitioners by developing a set of tools that introduces the fundamentals of environmental and economic analyses to regional practitioners and gives them the eye openers to better understand their region, the sectors in order to assess the priority and the potential for ‘circular’ policies.

To achieve this, 3 main tools were developed: the REAT tool an online calculation tool that performs environmentally extended input output analysis in the subnational regions (NUTS 2) of Europe; the Tools and Databases for Circular Economy tool that introduces available online tools to further expand the knowledge on Circular Economy; and the Industrial Symbiosis matchmaking tools that identifies possible uses for wastes as secondary raw materials at regional level.

Additionally, a set of educational material was developed to allow teaching such complex tools to practitioners. This report presents a summary of the educational material developed as well as a description of the practitioners that were subject to the material, through seminars, workshops, lectures and project work.

Educational Materials overview

The educational material developed considered the 3 main tools in the CircuMat project. It included not only an explanation, or exercises around the tools but also the presentation of the fundamental background knowledge that need to be understood to properly manipulate the CircuMat tool. All the educational materials were used in the ACE155 Urban Metabolism course of the MSc program Infrastructure and Environmental Engineering given at Chalmers. Adapted versions were made for the REAT tool and Industrial Symbiosis Matchmaking tool for the CircuMat workshops: Building a Regional Circular Economy Identifying regional waste flows as secondary inputs for the regional economy at TU Delft.

The REAT tool

The REAT tool is based on the concepts of Input-Output and performs economic and environmental calculations according to the Leontief model in the NUT2 regions of Europe. To present the tool two lectures were developed: the first on fundamentals of input-ouput and a simple exercise and the second on multiregional input-output and the REAT tool. Additionally, a tutorial about the REAT tool was developed and a project work description, that allowed for further exploration on the topic.

The Tools and Databases for Circular Economy tool

The Tools and Databases for Circular Economy tool is composed of information about several online services that provide tools and data for several types of methods that can be used under the Circular Economy concept. For example, MFA, LCA and System dynamics. To allow for exploration of this set of information a workshop format was developed which allow for the participants to research on a single tool or database and afterward present it in class. A brief document was created to guide the discussion.
